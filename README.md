SD SmartLearn 🎓

SD SmartLearn adalah platform pembelajaran interaktif berbasis web untuk siswa sekolah dasar yang menggabungkan belajar dan bermain (gamification learning).

Aplikasi ini membantu siswa belajar melalui kuis, permainan edukatif, sistem bintang, badge, dan unlock skin sehingga proses belajar menjadi lebih menyenangkan dan memotivasi.

2️⃣ Games Edukasi

Berbagai permainan edukatif tersedia untuk melatih kemampuan siswa:

Game	Deskripsi
Math Quiz	Latihan berhitung cepat
Acak Kata	Menyusun huruf menjadi kata
Benar / Salah	Menentukan pernyataan benar atau salah
Memory Card	Mencocokkan pasangan kartu
Tebak Gambar	Menebak objek dari gambar
Puzzle Angka	Menyusun angka menjadi urutan

3️⃣ Sistem Reward ⭐

Setiap jawaban benar akan memberikan bintang kepada siswa.

Bintang digunakan untuk:

membuka skin

meningkatkan progress belajar

mendapatkan badge

4️⃣ Badge Achievement 🏅

Siswa akan mendapatkan badge ketika menyelesaikan target tertentu.

Contoh badge:

Jago Matematika

Raja Membaca

Anak Pancasila

Anak Sehat

Seniman Hebat

Pejuang Belajar

5️⃣ Unlock Skin 🎨

Bintang yang dikumpulkan dapat digunakan untuk membuka skin karakter baru.

Skin	Syarat
Default Biru	Gratis
Space Ungu	5 ⭐
Jungle Hijau	2 ⭐
Candy Pink	12 ⭐

6️⃣ Profil Siswa 👤

Halaman profil menampilkan:

Avatar pengguna

Total bintang

Badge yang diperoleh

Progress belajar

7️⃣ Progress Belajar 📊

Sistem menampilkan progress siswa berdasarkan:

mata pelajaran

status penyelesaian
